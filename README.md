## Dependency

- Python2 or Python3
- Opencv-Python
- Flask

## Usage

- `git clone https://github.com/wangshub/Camera-Streamer-Py.git && cd Camera-Streamer-Py`
- `pip install -r requirements.txt`
- `python app.py`
- Open link [http://localhost:5000](http://localhost:5000) or `http://x.x.x.x:5000`
